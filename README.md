# MotionQR
MortionQRは。MVやクリエイティブ制作向けのQRコードアニメーション生成ツールです！

QRコードを時間経過に合わせて切り替え、MP4やWebMなどの動画やjpegやpngとして出力できます。
動画編集ソフトではなく、「QRアニメーション生成」に特化したツールです。

---

## 理論
理論を応用することでQRコードで様々なアニメーションを作成することができます。
QRコードは誤り訂正機能を備えており、一部が隠れたり装飾されたりしても認識できる場合があります。この特性を利用し、映像表現に応用することを目的としています。
